Express Scan Manager-Btype

Configures how the Express Scan Manager-Btype scans images from the printer and where the scan is saved.

Note
The Express Scan Manager-Btype settings apply to all scans and the settings are configured for each user.

You can configure the following fields:

Destination Path

Open the image file

Cancel button: Discards any changes and closes the dialog box.

OK button: Applies the changes and closes the dialog box.
