Destination Path

Specifies the path and name of the folder in which you want to save scanned data. You can also use the Browse... button to locate the path and name of the folder.

Note
The default is "/Users/User Name/Desktop".

Files names are saved in the following format: "img-YYMMDDHHMMSS-001.<extension>". The meanings of each field are as follows:

  • YY: The last 2 digits of the year when the file was created.

  • MM: The 2 digits representing the month when the file was created.

  • DD: The 2 digits representing the day when the file was created.

  • HH: The 2 digits representing the hour when the file was created.

  • MM: The 2 digits representing the minute when the file was created.

  • SS: The 2 digits representing the second when the file was created.

  • extension: The file format can be identified by file extension. Supported file formats are jpg, tif, and pdf.
